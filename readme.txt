Para configuração do e-mail é necessário a alteração do arquivo email_config.txt.
Onde:
    EMAIL_USER=emaildohost@gmail.com
    EMAIL_PASSWORD=senhadohost@gmail.com
    EMAIL_TO=seuemail@gmail.com

Para instalação no sistema operacional linux, no terminal:
Torne o script executável:
chmod +x install.sh

Execute o script como root:
sudo ./install.sh

Para instalação no sistema operacional linux:
Clique com o botão direito no arquivo install.bat e selecione Executar como administrador.
*O Python deve estar corretamente configurado no PATH do sistema, caso contrario especifique no TASK_COMMAND o caminho absoluto.
